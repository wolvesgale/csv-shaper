CSV整形（姓名分割・サンプル列準拠）

※ admin-customer-list-sample.csv を指定。未指定で取得失敗時は既定列を使用。

漢字＋英字 → 英字のみ削除（スペースは保持） 最初の半角スペースのみ分割（右側の空白は保持） 全角スペースも分割対象 スペース無しは推定分割

仕様（クリックで展開）
  • サンプル列に完全準拠（無い列は破棄／足りない列は空欄補完）。
  • 氏名は最初の半角スペースで分割（オプションで全角も）。
  • 漢字＋英字混在は英字のみ除去してから分割（スペースは維持）。
  • ヘッダ無しCSVにも対応：左端（__col0）を氏名、3列目（__col2）を会社名フォールバック。
  • 名前が FALSE/false/N/A/-/空 や 【未入力】 相当なら、姓・名ともに【未入力】。
  • 出力が 2,500 行を超える場合は 2,500 行ごとに自動分割して複数CSVを連続ダウンロード。
No server / static. Deploy anywhere.
